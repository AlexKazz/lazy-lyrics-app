## Getting Started

Getting started is easy! Just follow the steps below to get a local copy up and running.

### Prerequisites

- npm
  ```sh
  npm install
  ```

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:AlexKazz/lazy-lyrics-app.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the server
   ```js
   npm run start
   ```
4. Open the app in your browser
   ```sh
    http://localhost:3000/
   ```
